# CSF_Assessment
NIST CSF 2.0 Assessment Tool - A single standalone HTML file (no dependencies)<br>
<br>
<b>Features</b>
- Assessment Wizard<br>
- All 6 Functions, 22 Categories, and 106 Subcategories from the official NIST CSF 2.0 framework<br>
- Function sidebar with completion counts (e.g., "5/31" for GOVERN)<br>
- Expandable/collapsible category cards<br>
- Per-subcategory Current Tier and Target Tier selectors (N/A, T1-T4)<br>
- Notes/evidence field per subcategory<br>
- Progress bar tracking overall completion<br>
- Default target tier selector for gap analysis<br>
- Organization name, assessor, date, and scope fields<br>
<br>
<b>Heat Map Visualization</b><br>
- Color-coded cells for every subcategory organized by function/category<br>
- Hover tooltips showing subcategory descriptions and current tier<br>
- Legend for all tier levels<br>
<br>
<b>Dashboard</b><br>
- Summary score cards per function with bar charts<br>
- Overall composite score<br>
- Radar chart showing current vs target tier overlay across all 6 functions<br>
<br>
<b>Gap Analysis</b><br>
- Table of all subcategories where current < target, sorted by gap size<br>
- Color-coded gap severity badges (red/yellow/blue)<br>
<br>
<b>Excel Export</b><br>
- Exports 3-sheet Excel workbook (XML Spreadsheet format)<br>
- Assessment Details - all 106 subcategories with scores and notes<br>
- Summary Scores - function and category averages<br>
- Gap Analysis - sorted gaps with color highlighting<br>
<br>
<b>Save/Load</b><br>
- Save progress to JSON file, reload later to continue
